# Curve Fever Enhanced

## A few words of introduction:
Remake of a popular web browser game made in Unity & C#

## Game description:
You are playing as a "snake" that competes with others to conquer the whole map.

## Controls:
* **A** or **←** - turn left
* **D** or **→** - turn right
